# ChatRoom - 多人在线聊天室（服务端 + 客户端）

本项目是一个基于 C++ 开发的局域网聊天室系统，支持多用户同时在线，包含用户注册/登录、公开/私有房间创建、房间密码保护、实时消息聊天等功能。服务端使用 Winsock2 和多线程处理客户端连接，数据存储于 MySQL 数据库；客户端使用 Qt 6 框架提供图形界面。

## 功能特性

### 服务端
- 支持多个客户端同时连接（多线程）
- 用户认证：登录、注册（密码使用 SHA256 加密存储）
- 房间管理：
  - 创建房间（公开/私有，私有房间可设置密码）
  - 加入房间（私有房间需验证密码）
  - 离开房间、自动清理空房间（大厅除外）
  - 房间列表实时同步
- 聊天消息：房间内成员实时收发消息（大厅不支持聊天，仅作为入口）
- 数据库持久化：用户、房间、成员关系均保存到 MySQL
- 自动加载数据库中已有的房间列表

### 客户端（Qt 图形界面）
- 认证对话框：登录 / 注册切换
- 大厅窗口：
  - 显示所有房间列表（房间名、类型、人数）
  - 创建房间（输入房间名、选择公开/私有、私有房间需设密码）
  - 加入房间（私有房间需输入密码）
- 聊天窗口：
  - 实时显示房间内消息
  - 发送消息
  - 主动退出房间
- 自动重连 / 错误提示

## 技术栈

| 组件     | 技术                                                         |
| -------- | ------------------------------------------------------------ |
| 服务端   | C++17，Winsock2（Windows Socket），MySQL C API，多线程（std::thread） |
| 客户端   | Qt 6（C++17），QTcpSocket，QWidget 界面                      |
| 数据库   | MySQL 5.7+ / 8.0+，使用 `utf8mb4` 字符集支持中文             |
| 构建工具 | 服务端：MinGW / MSVC 命令行或 IDE；客户端：Qt Creator（qmake 或 CMake） |

## 环境要求

### 通用要求
- Windows 操作系统
- MySQL 数据库服务

### 服务端
- 编译器：支持 C++17（MinGW-w64、MSVC 2019+）
- MySQL Connector/C（即 `libmysql.dll` 和头文件 `mysql.h`）
- 链接库：`libmysql.lib` / `libmysql.dll.a`，`ws2_32`

### 客户端
- Qt 6.2 或更高版本（MinGW 或 MSVC 编译套件）
- 编译器支持 C++17
